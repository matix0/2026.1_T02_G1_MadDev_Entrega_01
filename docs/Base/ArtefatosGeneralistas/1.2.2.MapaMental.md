# 1.2.2 - Mapa Mental
## O que é o Artefato Mapa Mental?
O Artefato Mapa Mental é um artefato visual que representa de forma hierárquica e interconectada as ideias, conceitos, tarefas ou informações relacionadas a um determinado tema. Ele é utilizado para organizar e estruturar o pensamento, facilitando a compreensão e a memorização de informações complexas. O Mapa Mental é composto por um nó central, que representa o tema principal, e ramos que se estendem a partir desse nó, representando subtemas, ideias ou tarefas relacionadas. Cada ramo pode conter palavras-chave, imagens ou símbolos que ajudam a ilustrar as conexões entre os diferentes elementos do mapa.
